Project Objective:
The objective of this project is to analyze supply chain operations using the DataCo Smart Supply Chain dataset and build a dimensional data model in Power BI.
Data set :https://www.kaggle.com/datasets/shashwatwork/dataco-smart-supply-chain-for-big-data-analysis 

Dataset Description:
The project uses the DataCo Smart Supply Chain for Big Data Analysis dataset from Kaggle.
-Customer information
-Product details
-Orders
-Shipping information
-Delivery status
-Geographic locations
-Sales and profit
-Departments and categories

Data Model Overview:
The project follows a Star Schema data model.
-Fact Table
Fact_table
-Dimension Tables
1.Dim_Customer
2.Dim_Product
3.Dim_Category
4.Dim_Department
5.Dim_Shipping
6.Dim_Location
7.Dim_Date

Project Outcome:
-Designed a clean Star Schema data model.
-Created multiple dimension tables from a single transactional dataset.
-Built a reusable Date dimension for time-based analysis.
